Keyboard shortcuts:
   Shift: Switch to draw mode. Hold shift and hover over districts to add them to the selected nation.
   Control: Switch to erase mode. Hold control and hover over districts to delete them from the selected nation.
   `: (On the top-left corner of the keyboard) Switch to browse mode. Does not allow adding or deleting districts from nations.

   [: Switch to district mode. Select by district.
   ]: Switch to province mode. Select by province.
   \: Switch to country mode. Select by country.

   Map name: Click on Map 1 and type to change the map name.
   Nation name: Click on a nation name (e.g. "Nation 1") and type to change the nation's name.
   Right-click on nation button: Allows the user to change the color of the nation.
